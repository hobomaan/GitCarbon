# Contributing to GitCarbon

Thank you for your interest in making GitCarbon better! This repository is the **public issue tracker and discussion hub** for the GitCarbon VS Code extension. The extension source code is not open-source at this time.

---

## How you can contribute

### Report bugs

Found something broken? [Open a bug report →](https://github.com/hobomaan/GitCarbon/issues/new?template=bug_report.md)

Good bug reports include:
- A clear, descriptive title
- Steps to reproduce the issue
- Expected vs. actual behaviour
- Your environment (OS, VS Code version, Git version, GitCarbon version)
- Screenshots or screen recordings when helpful

### Request features

Have an idea? [Open a feature request →](https://github.com/hobomaan/GitCarbon/issues/new?template=feature_request.md)

Great feature requests include:
- A description of the problem or workflow gap
- Your proposed solution
- How it fits into your daily workflow
- References to similar features in other tools

### Join discussions

Use [GitHub Discussions](https://github.com/hobomaan/GitCarbon/discussions) (when enabled) to:
- Ask questions about using GitCarbon
- Share tips and workflows
- Suggest ideas that aren't fully formed yet

---

## Guidelines

- **Search first** — check if your issue or idea already exists before opening a new one.
- **One issue per report** — keep bug reports and feature requests focused on a single topic.
- **Be respectful** — follow our [Code of Conduct](CODE_OF_CONDUCT.md).
- **Be specific** — vague reports are hard to act on. Include versions, steps, and screenshots.

---

## What we can't accept

Since the extension source is closed-source, we cannot accept pull requests for extension code. However, contributions to this repo's documentation (README, templates, etc.) are welcome via pull request.

---

## Security

If you discover a security vulnerability, please **do not** open a public issue. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

---

Thank you for helping make GitCarbon better for everyone!
